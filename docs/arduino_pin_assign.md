# Arduino Pin assign Interface

## Trig(距離センサ Trig信号)
pin-12

## Echo(距離センサ Echo信号)
pin-13

## SG90(サーボモータ)
pin-10

## Vp0(フォトセンサ HC14出力)
pin-4

## Vp1(フォトセンサ HC14出力)
pin-9

## M2_IN1(左モータ方向制御)
pin-3

## M2_IN2(左モータ方向制御)
pin-5

## M1_IN1(右モータ方向制御)
pin-11

## M1_IN2(右モータ方向制御)
pin-6

## DA1(LED/警報信号)
pin-7
